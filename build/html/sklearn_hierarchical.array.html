sklearn_hierarchical_classification.array Module¶

Helpers for workings with sequences and (numpy) arrays.

sklearn_hierarchical_classification.array.apply_along_rows(func, X)¶

Apply function row-wise to input matrix X. This will work for dense matrices (eg np.ndarray) as well as for CSR sparse matrices.

Parameters:
X : (sparse) array-like, shape = [n_samples, n_features]

Data.

y : list-of-lists - [n_samples]

For each sample, y maintains list of labels this sample should be used for in training.
Returns:
X_, y_

Transformed by ‘flattening’ out y parameter and duplicating corresponding rows in X
